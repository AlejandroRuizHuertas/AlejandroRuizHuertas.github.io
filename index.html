Gestor de base de datos COVID-19

El motivo de este proyecto fue emular una base de datos espacio-temporal. Para ello, se cogió la idea del COVID-19 y se hizo un programa que permitiera ver el avance de los casos a lo largo del tiempo y del país. Se usó Java como lenguaje principal y MySQL como el Sistema Gestor de Base de Datos.

  • Fecha: Abril 2020
  • Cliente: Gestión de Sistemas de Información

Gestión de circuitos turísticos de Ciudad Real

Este proyecto ofrece una interfaz interactiva en móvil para un enunciado hipotético sobre la gestión de circuitos turísticos de la provincia de Ciudad Real. Está desarrollado en Flutter con lenguaje Dart, concretamente para el entorno de Android, aunque podría ser exportada perfectamente para iOS. Pese a no ofrecer una funcionalidad totalmente operativa, está pensado para una aplicación 100% funcinal con sólo añadirle lógica y persistencia.

  • Fecha: Julio 2020
  • Cliente: Interacción Persona-Ordenador II

Gestor de descarga de ficheros mediante middleware

Este proyecto está destinado para aplicar lo aprendido en la asignatura de Sistemas Distribuidos. Concretamente, con la tecnología Zero-C Ice, aplicar el patrón de diseño Factoría para gestionar una serie de "Downloaders" que irán descargando archivos a medida de que se vayan recibiendo peticiones de descarga.

  • Fecha: Enero 2020
  • Cliente: Sistemas Distribuidos
