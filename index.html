Gestión de circuitos turísticos de Ciudad Real

  • Descripción: Este proyecto ofrece una interfaz interactiva en móvil para un enunciado hipotético sobre la gestión de circuitos turísticos de la provincia de Ciudad Real. Está desarrollado en Flutter con lenguaje Dart, concretamente para el entorno de Android, aunque podría ser exportada perfectamente para iOS.
  • Tecnologías: Flutter, Dart
  • Enlace: Github

Gestor de base de datos COVID-19

  • Descripción: El motivo de este proyecto fue emular una base de datos espacio-temporal. Para ello, se cogió la idea del COVID-19 y se hizo un programa que permitiera ver el avance de los casos a lo largo del tiempo y del país.
  • Tecnologías usadas: Java, MySQL
  • Enlace: Github

Gestor de descarga de ficheros mediante middleware

  • Descripción: Con la tecnología Zero-C Ice, se aplica el patrón de diseño Factoría para gestionar una serie de "Downloaders" que irán descargando archivos a medida de que se vayan recibiendo peticiones de descarga.
  • Tecnologías: ZeroC-Ice
  • Enlace: Github
